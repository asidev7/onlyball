Provably fair

OnlyBall runs no smart contract, so trust comes from math instead: a commit-reveal scheme combined with an unpredictable public Solana blockhash (used only as a random beacon, never for moving funds) decides every winner. The server commits to a secret seed before you can buy a single ticket, and reveals it only after the drawing closes — anyone can recompute the result and confirm nothing was changed after the fact.
